annotations
Create - принимает id задачи, токен и массив аннотаций, возвращает результат выполнения команды, true/false соотвественно и ошибку
Delete - принимает id аннотации и токен, возвращает результат работы функции и ошибку 
FetchAll - принимает id задачи, токен и возвращает массив аннотаций с ошибкой
Update - принимает id аннотации, токен и обновленную аннотацию, возвращает результат работы функции true/false и ошибку
